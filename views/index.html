URL Shortener

This is a URL shortener where you enter a valid URL into the input, and you'll be returned a JSON object, which will have the URL you inputted and a new URL, the new URL can appended to '/api/shorturl/' to redirect you to the original url.

For example '/api/shorturl/5' will redirect you to github.

If you input an invalid URL you'll be returned an error stating your URL is invalid.

Please enter a URL in the format 'https://www.domain.com'.
